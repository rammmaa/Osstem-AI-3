## 처음 시작하기 — "이 순서대로 따라하세요"

### 1단계: 프로젝트 받기 (처음 한 번만)

```bash
git clone https://github.com/rammmaa/Osstem-AI-3.git
cd Osstem-AI-3
```

> 이 명령어는 GitHub에 있는 프로젝트를 **내 컴퓨터로 복사**하는 것입니다.

### 2단계: 파이썬 가상환경 만들기

```bash
python -m venv venv
```

그다음 가상환경을 켭니다:

- **Mac/Linux**: `source venv/bin/activate`
- **Windows**: `venv\Scripts\activate`

> 가상환경은 **이 프로젝트 전용 작업공간**입니다. 다른 프로젝트와 섞이지 않게 해줍니다.

### 3단계: 필요한 도구 설치

```bash
pip install -r requirements.txt
```

> 요리로 치면 **재료 준비**입니다. 프로젝트에 필요한 도구들을 한 번에 설치합니다.

### 4단계: 비밀번호(API 키) 설정

```bash
cp .env.example .env
```

그다음 `.env` 파일을 열어서 본인의 API 키를 입력합니다.

> `.env.example`은 **양식**이고, `.env`는 **실제 비밀번호가 들어가는 파일**입니다.
> `.env` 파일은 절대 GitHub에 올리지 않습니다!

| 해야 할 것 | 하면 안 되는 것 |
