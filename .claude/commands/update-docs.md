CLAUDE.md를 현재 프로젝트 상태에 맞게 업데이트해 주세요.

## 절차

1. 현재 프로젝트 상태 파악
   - `git log --oneline -10`으로 최근 커밋 확인
   - `git diff HEAD~5 --stat`으로 최근 변경된 파일 목록 확인
   - 프로젝트 루트의 폴더/파일 구조 확인 (`ls`)
   - `.claude/` 디렉토리 내 파일 확인
   - `.claude/commands/` 디렉토리의 슬래시 커맨드 목록 확인

2. `.claude/CLAUDE.md` 파일 읽기

3. 아래 섹션들을 현재 상태에 맞게 업데이트
   - **구조**: 새로운 폴더나 파일이 추가되었으면 반영
   - **문서 현황**: 새로운 문서가 추가되었으면 표에 추가
   - **Claude Code 설정 (.claude/)**: 새로운 설정 파일이나 커맨드가 추가되었으면 반영
   - **슬래시 커맨드**: 새로운 커맨드가 추가되었으면 표에 추가
   - **진행 현황**: 오늘 날짜로 새로운 작업 내역 추가 (날짜 형식: YYYY-MM-DD)

4. 변경된 내용을 간단히 요약해서 알려주세요

## 주의사항

- 기존 내용을 삭제하지 마세요 (진행 현황은 누적 기록)
- 날짜는 반드시 절대 날짜로 기록 (예: 2026-03-27)
- 추측하지 말고 실제 파일/폴더를 확인한 내용만 반영하세요
